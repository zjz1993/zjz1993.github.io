jz-ui组件库使用说明

jz-ui组件库

设计初衷

jz-ui是自己在学习了vue2.0框架后，想提高自己使用vue的水平，恰巧在实习的公司又开发了vue的前端项目，所以想把常用的ui功能集合到一起
github地址：https://github.com/zjz1993/jz-ui
使用方法：在vue工程的main.js里，先import jzUi from ‘jz-ui’ 随后import ‘jz-ui/lib/index.css’，最后Vue.use(jzUi)
目前开发完成的功能有button组件和icon组件，其余组件还在开发中~~

API

属性 说明 用法 默认值
type 图标的名称，从fontAwesome获得 <jzIcon type="bath" /> 无
size 图标的大小 <jzIcon type="bath" size="30px" /> 默认图标的大小是width=15 height=15
color 图标的颜色 <jzIcon type="bath" size="30px" color="red"/> 支持颜色的引文名称，rgb颜色格式，十六进制颜色格式等命名方式
